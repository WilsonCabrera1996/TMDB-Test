# Cine APP

## Get Started

- Install dependencies
```bash
npm install
```

- Run the project
```bash
npm run dev
```

### If you are gonna use a different package manager instead of `npm`, just delete the package-lock.json and run the correct command:

· Using pnpm:
```bash
pnpm install
```

· Using yarn:
```bash
yarn install
```

· Using bun:
```bash
bun install
```
